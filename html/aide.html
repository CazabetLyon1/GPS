Comment utiliser la carte ?

  • Ces deux boutons permettent de changer le mode de la souris. Pour passer dans le mode : déplacement de point, il faut cliquer sur le deuxième bouton (Celui avec la main). Le premier bouton permet de remettre la souris dans son état par défaut ce qui permet de redéplacer la carte à nouveau.

  • Ce bouton permet de rééchantillonner le fichier, c'est-à-dire d'enlever un certain nombre de points. Le champ de saisie situé au-dessus du bouton permet de choisir le nombre de données à enlever.

  • Ce bouton permet d'annuler la dernière action effectuée.

  • Ce bouton permet d'annuler l'annulation de la dernière action effectuée.

  • Ce bouton permet d'ajouter un point à partir de la fin de la trace GPS sélectionnée.

  • Cette option permet la suppression du point sélectionné.

  • Cette fonctionnalité fusionne deux traces en fonction du côté souhaité.

  • Ce bouton permet de couper en deux morceaux une trace à partir du point sélectionné.

  • Une fois l'édition de votre trace, ce bouton permet de télécharger le nouveau fichier.
